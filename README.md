# Codex Crypto Cycle Dashboard

A public comparison dashboard for the Codex posterior fork of the BTC / ETH
cycle model.  It keeps the legacy score cards visible, but leads with a
future-low posterior: price and timing quantiles, further-downside
probabilities, and a posterior-adjusted deployment target.

**Live:** https://attm-agenticcoding.github.io/crypto-cycle-dashboard-codex/

Only model outputs, market prices, and percentages are shown — no dollar amounts.
Research / educational only — not investment advice.
